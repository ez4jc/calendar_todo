# Calendar Todo - 日历清单

统信UOS上的日历清单软件，支持系统托盘、半透明日历、待办事项管理。

![Platform](https://img.shields.io/badge/platform-UOS%20%7C%20Linux%20ARM64-blue)
![Qt](https://img.shields.io/badge/Qt-5.11.3-green)
![License](https://img.shields.io/badge/license-MIT-orange)

## 功能特性

| 功能 | 说明 |
|------|------|
| 系统托盘 | 启动最小化到托盘，右键菜单，双击显示 |
| 半透明窗口 | 默认40%透明度，可调节10%-100% |
| 无边框窗口 | 可拖拽移动，边缘调整大小 |
| 日历网格 | 显示当月日期，上下文日期灰色显示 |
| 待办事项 | 双击添加，完成切换（删除线） |
| 本地存储 | SQLite数据库，位置记忆 |

## 编译构建

### 环境要求

- Qt 5.11+
- C++11
- SQLite3
- GCC 8+

### 编译步骤

```bash
# 克隆项目
git clone https://github.com/ez4jc/calendar_todo.git
cd calendar_todo

# 生成Makefile
qmake calendar_todo.pro

# 编译
make -j$(nproc)

# 运行
./calendar_todo
```

## 使用说明

| 操作 | 说明 |
|------|------|
| 双击日期 | 打开待办对话框 |
| 输入标题保存 | 添加新待办 |
| 点击待办列表项 | 切换完成/编辑/删除 |
| 拖拽窗口 | 移动位置 |
| 拖拽窗口边缘 | 调整大小 |
| 点击×按钮 | 最小化到托盘 |
| 托盘右键 | 显示窗口或退出 |

## 数据存储

- 数据库：`~/.config/calendar_todo/data.db`
- 窗口设置：透明度、位置、尺寸
